h1. Chrome WebStore Install Notification Bar

Displays a notification bar informing users to install the Chrome Web Store App.

More info "here".

h2. Requirements:

* Chrome Web Browser

h2. How To:

* Copy the chrome-notification.*css* files to web server
* Include the chrome-notification.js script on your main page
* Edit the configuration parameters at the top of the chrome-notification.js
