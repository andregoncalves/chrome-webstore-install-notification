h1. Chrome Web Store Notification Bar

Javascript library to dynamically display a notification bar redirecting Chrome users to the Web Store install page.
Example:

More info "here":http://andregoncalves.com/2011/07/18/chrome-webstore-notification-bar.html.

h2. Requirements:

* Chrome Web Browser

h2. How To:

* Copy the chrome-notification.* files to web server
* Include the chrome-notification.js script on your main page
* Edit the configuration parameters at the top of the chrome-notification.js

h2. Inspired by:

* "badgemator":https://github.com/PaulKinlan/badgemator by Paul Kinlan
* "jolicloud":http://www.jolicloud.com/ notification bar
